# Technical Design Source

The implementation is based on `PocketTrainer_Technical_Design.md` supplied by the project owner on 2026-07-16. The source document remains outside the repository at `/home/xavrir/Downloads/PocketTrainer_Technical_Design.md`.

The MVP narrows the broader design to movement intelligence, deterministic training, progression, offline operation, identity/privacy, and the minimum backend needed to prove that loop. Supporting nutrition, health, gym, and public-social domains are deliberately deferred.
