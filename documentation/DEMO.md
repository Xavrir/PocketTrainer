# Hackathon Demo Script

Target length: 3–4 minutes.

1. **Problem (20s):** workout videos cannot see the beginner performing the movement.
2. **Path (25s):** show the personalized Strength/Yoga/Mobility curriculum and dual XP/mastery locks.
3. **Privacy (15s):** enable airplane mode before opening the lesson; explain that frames remain native.
4. **Calibration (25s):** demonstrate distance/orientation/full-body guidance.
5. **Coaching (60s):** complete a squat set with one correction, visible tracking confidence, rep state, and recovery after leaving frame.
6. **Result (30s):** show transparent form/completion/control/consistency breakdown and difficulty/pain check.
7. **Progression (20s):** show XP preview, mastery requirement, and why a harder lesson remains gated when unsafe.
8. **Sync (20s):** restore connectivity and show exactly-once server confirmation.
9. **Architecture (20s):** close with on-device inference, Azure product data, Supabase identity, and Cloudflare content.

Never claim professional validation, target device performance, or cloud deployment until the corresponding checkpoint evidence exists.
